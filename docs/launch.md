# VoxKey launch materials

## 30-second demo

Record at 1080p with notification popups disabled and no personal content visible.

1. **0-4 seconds:** Show the VoxKey name and caption: `Private offline voice typing for Windows`.
2. **4-10 seconds:** Focus Notepad, hold Right Ctrl, and show only the small animated orb while saying: `VoxKey keeps dictation on my computer.`
3. **10-15 seconds:** Release the key and show the polished sentence appear.
4. **15-21 seconds:** Open settings, switch the hotkey to F8, and briefly show `Ready` plus the microphone selector.
5. **21-26 seconds:** Dictate once into a browser text field using F8.
6. **26-30 seconds:** Show: `No account. No telemetry. No subscription. Open source.` followed by the GitHub repository name.

Do not show first-run download time in the short demo. Publish a separate setup clip if testers need it.

>
> VoxKey records only while a selected hotkey is held, transcribes speech locally, polishes the text with a private local writing model, and pastes it back into the original application. It does not require an account, subscription, telemetry, or cloud dictation service.
>
> I am looking for Windows beta testers across different CPUs, GPUs, microphones, and applications. Source code and releases are available at:
