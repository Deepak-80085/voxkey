# VoxKey v0.1.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a GitHub-ready Windows v0.1.0 of VoxKey with a native Qt tray/settings shell, Siri-inspired transient dictation HUD and sounds, while retaining the verified local dictation pipeline.

**Architecture:** Add a small UI event contract between the existing hotkey/controller pipeline and a Qt application shell. The pipeline publishes immutable state/timing events without importing Qt; Qt owns the tray, overlay, sound cues, settings, and orderly process exit. The existing `small.en` + local Ollama pipeline, target HWND restoration, strict no-raw-paste policy, and single-instance behavior remain intact.

**Tech Stack:** Python 3.11+, PySide6, Qt Multimedia, pynput, sounddevice, faster-whisper/CTranslate2, Ollama HTTP API, PyInstaller, Inno Setup, unittest.

## Global Constraints

- Windows only; store all mutable state beneath `%LOCALAPPDATA%\VoxKey`.
- English-only `small.en`; never select, download, or fall back to `base.en`.
- Required local writer model: `qwen3.5:0.8b`; never paste an unpolished/raw transcript.
- Dictation trigger remains hold Right Ctrl for at least `0.28` seconds.
- HUD must never take focus, accept input, or delay capture/transcription/paste.
- Sound feedback is on by default and failures are non-fatal.
- One normal-process instance; elevated apps show a clear limitation instead of requesting elevation by default.
- v0.1.0 excludes onboarding, autostart, vocabulary editor/history UI, arbitrary remapping, and per-app controls.
- Keep original `E:\simplespeech` and its unrelated `index.html` untouched.

```python
# voxkey_events.py
from dataclasses import dataclass
from queue import Empty, SimpleQueue
from voxkey_runtime import AppState

@dataclass(frozen=True)
class UiEvent:
    kind: str
    state: AppState | None = None
    detail: str | None = None
    elapsed_ms: int | None = None

class EventBus:
    def __init__(self):
        self._events = SimpleQueue()
    def publish(self, event: UiEvent) -> None:
        self._events.put(event)
    def drain(self) -> list[UiEvent]:
        result = []
        while True:
            try:
                result.append(self._events.get_nowait())
            except Empty:
                return result
```
